
# Novo Volkswagen Logus 2025 - Site de Lançamento

Este pacote contém a landing page do Novo VW Logus 2025, com galeria de imagens reais, visual moderno e responsivo, inspirado no site oficial da Volkswagen.

## Estrutura do Projeto

- `logus_site_final.html`: página principal do site.
- `imagens/`: pasta contendo todas as imagens utilizadas na galeria.

## Como Visualizar

1. Extraia o conteúdo do arquivo .zip.
2. Abra o arquivo `logus_site_final.html` com um navegador (Chrome, Firefox, Edge).
3. As imagens já estão vinculadas corretamente e a página funcionará offline.

## Como Publicar Online

### Opção 1: GitHub Pages
1. Crie um repositório no GitHub.
2. Envie todos os arquivos do projeto (HTML + pasta `imagens/`).
3. Vá em "Settings" > "Pages" e selecione a branch principal.
4. O site ficará disponível em: `https://seuusuario.github.io/nome-do-repo`.

### Opção 2: Hospedagem Gratuita com Netlify
1. Acesse [https://www.netlify.com](https://www.netlify.com)
2. Crie uma conta e clique em "Add New Site".
3. Faça o upload da pasta com os arquivos.
4. O site será publicado automaticamente.
